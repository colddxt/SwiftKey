The procedures to build this application:

1. Create a new Java project in an IDE such as Eclipse.

2. Import all java files in the src folder into the src folder of your project.

3. Put the "wordf.txt" file in the project folder, side by side with the src folder. (You could also modify line 20 of Dictionary.java. You should change "./wordf.txt" to the path of wordf.txt on your computer).

4. Compile and run Keyboard.java.

You could also run this application in command line.
